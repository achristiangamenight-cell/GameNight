Find Google Form Entry IDs

Method 1: Using Browser Console

  1. Open the Google Form: Open Form
  2. Press F12 (or Right-click → Inspect) to open Developer Tools
  3. Go to the Console tab
  4. Paste and run this code:
// Find all entry IDs in the form const inputs = document.querySelectorAll('input[name^="entry."], textarea[name^="entry."], select[name^="entry."]'); const entryIds = {}; inputs.forEach(input => { const name = input.name; const label = input.closest('.freebirdFormviewerViewItemsItemItem')?.querySelector('.freebirdFormviewerViewItemsItemItemTitle')?.textContent || input.closest('[role="listitem"]')?.querySelector('[role="heading"]')?.textContent || 'Unknown Field'; entryIds[name] = label.trim(); }); console.log('Entry IDs found:'); console.table(entryIds); // Copy to clipboard const output = Object.entries(entryIds).map(([id, label]) => `${id}: ${label}`).join('\n'); navigator.clipboard.writeText(output).then(() => { console.log('Entry IDs copied to clipboard!'); alert('Entry IDs copied to clipboard! Check the console for details.'); });

Method 2: Manual Inspection

  1. Open the Google Form in your browser
  2. Right-click on the first question and select "Inspect"
  3. Look for input elements with names like name="entry.123456789"
  4. Note down each entry ID for each question in order

Method 3: View Page Source

  1. Open the Google Form
  2. Press Ctrl+U (or Right-click → View Page Source)
  3. Press Ctrl+F and search for "entry."
  4. You'll find entries like name="entry.XXXXXXX"
